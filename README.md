# Diabetes Prediction using Random Forest

## 📌 Overview
This project focuses on predicting diabetes in patients based on medical diagnostic measurements. I used the **Pima Indians Diabetes Database** and implemented a Machine Learning pipeline to classify patients as healthy or diabetic.

## 🚀 Key Features
- **Data Analysis:** Performed statistical analysis using Pandas.
- **Model Optimization:** Tested different Decision Tree sizes to prevent overfitting.
- **Ensemble Learning:** Implemented a **Random Forest Regressor** for better stability and accuracy.
- **Diagnostics:** Used a **Confusion Matrix** to evaluate True Positives and False Negatives (critical in medical AI).

## 📊 Results
- **Accuracy:** ~80.5% on the validation set.
- **Top Predictors:** Glucose levels and BMI were identified as the most significant features.
- **Medical Reliability:** Successfully reduced missed cases (False Negatives) by optimizing feature selection.

## 🛠 Tech Stack
- **Python** (Pandas, Scikit-learn)
- **Machine Learning:** Random Forest, Decision Trees
